Extra Packages repo for OpenWrt 24.10 by Marvel Dronnis


Данный репозиторий имеет мои пакеты и некоторые пакеты собранные от сообщества с разных git репозиториев.



Установка на OpenWrt


1. Добавить ключ репозитория в хранилище ключей opkg:

wget https://opkg.oldlan.ru/key-build.pub

opkg-key add key-build.pub

2. Добавить репозиторий в список репозиториев:

echo "src/gz openwrt_extras https://opkg.oldlan.ru/openwrt-24.10/$(. /etc/openwrt_release ; echo $DISTRIB_ARCH)" >> /etc/opkg/customfeeds.conf

3. Обновите списки репозиториев:

opkg update

4. Теперь вы можете перейти в luci и устанавливать пакеты. После установки пакетов luci-app-* вам может понадобиться reboot.


Все шаги в одну комманду:

wget https://opkg.oldlan.ru/key-build.pub && opkg-key add key-build.pub && echo "src/gz openwrt_extras https://opkg.oldlan.ru/openwrt-24.10/$(. /etc/openwrt_release ; echo $DISTRIB_ARCH)" >> /etc/opkg/customfeeds.conf && opkg update

1. Add new opkg key:

wget https://opkg.oldlan.ru/key-build.pub

opkg-key add key-build.pub

2. Add opkg repository:

echo "src/gz openwrt_extras https://opkg.oldlan.ru/openwrt-24.10/$(. /etc/openwrt_release ; echo $DISTRIB_ARCH)" >> /etc/opkg/customfeeds.conf

3. Update opkg sources:

opkg update

4. Now you can go to luci and search for packages to install. After the first installed, you need to reboot.
